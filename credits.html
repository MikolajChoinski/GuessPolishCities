Inspiration for the project

The project was inspired by the cities quizzes available at the CityQuiz.io, a website developed and operated by Ian Fisher. I couldn't wait for the polish edition of the quiz so I decided to take the matters into my own hands and created this site.

Software

  • The frontend is written in JavaScript and styled in vanilla CSS.
  • The project is bundled using webpack.
  • I used the Leaflet.js library to create an interactive map and markers.
  • I used the Map pin icon.svg created by Mono to style the map markers.
  • I used the MediaWiki Action API to fetch the data about the population of Poland and about its cities and villages.

    • Data

  • All data is taken from Wikidata unless otherwise noted.
  • The map is taken from OpenStreetMap and CARTO
